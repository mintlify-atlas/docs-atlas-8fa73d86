## Introduction

The PPG CLI provides a comprehensive set of commands to manage your desktop application's entire lifecycle - from project initialization to building production-ready installers. All commands are invoked using `ppg <command>` in your terminal.

## Common Workflows

### Starting a New Project

```bash
# Create a new project
ppg init

# Run it in development mode
ppg start
```

### Development Cycle

```bash
# Create new components as you build
ppg create component

# Test your changes
ppg test

# Run the app to see changes
ppg start
```

### Building for Production

```bash
# Clean previous builds
ppg clean

# Compile to executable
ppg freeze

# Create installer
ppg installer
```

## Command Line Parser

PPG's CLI is built on Python's `argparse` module with custom command registration through the `@command` decorator. Each command is automatically discovered and made available when you run `ppg`.

### Getting Help

You can get help for any command:

```bash
# General help
ppg --help

# Command-specific help
ppg freeze --help
```

## Command Categories

### Project Management
- `init` - Initialize new projects
- `version` - Check PPG version

### Development
- `start` - Run from source
- `create` - Generate code scaffolding
- `test` - Run tests

### Build & Distribution
- `freeze` - Compile executable
- `installer` - Create installers
- `clean` - Clean build artifacts
