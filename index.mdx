  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-lg text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with PPG in minutes
  </p>

  <Steps>
    <Step title="Install PPG">
      Install PPG using pip in your Python environment:
      
      ```bash
      pip install ppg
      ```
      
      PPG supports Python 3.8+ and works with PyQt5, PyQt6, PySide2, and PySide6.
    </Step>
    
    <Step title="Initialize Your Project">
      Create a new PPG project with the interactive CLI:
      
      ```bash
      ppg init
      ```
      
      The CLI will guide you through project configuration, including app name, Qt binding selection, and bundle identifier setup.
    </Step>
    
    <Step title="Run Your Application">
      Start your app in development mode with hot reloading:
      
      ```bash
      ppg start
      ```
      
      Your application window will open, and any changes you make to your code will be reflected instantly.
    </Step>
    
    <Step title="Build and Package">
      When ready to distribute, compile your app to a standalone executable:
      
      ```bash
      ppg freeze
      ppg installer
      ```
      
      PPG automatically creates platform-specific installers for Windows, macOS, and Linux.
    </Step>
  </Steps>
</div>
